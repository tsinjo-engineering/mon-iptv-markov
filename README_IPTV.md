# Ma Liste IPTV — Gratuite et Legale

Liste M3U personnelle avec 44 entrees : chaines directes + sources indexees mondiales.
Usage personnel uniquement. Partage gratuit autorise.

---

## Utiliser dans VLC

1. Ouvrir VLC
2. Menu : Media > Ouvrir un flux reseau
3. Coller l'URL de ta liste GitHub Pages :

   https://TON-USERNAME.github.io/mon-iptv/ma_liste_complete.m3u

4. Cliquer Lire

---

## Utiliser dans TiviMate / IPTV Smarters (Android TV)

1. Ajouter une playlist
2. Choisir "URL / Remote"
3. Coller la meme URL
4. Valider — les chaines se chargent automatiquement

---

## Utiliser dans Kodi

1. Installer l'addon : PVR IPTV Simple Client
2. Dans sa configuration, coller l'URL M3U
3. Activer et redemarrer Kodi

---

## Mettre en ligne sur GitHub (1 seul lien pour tout)

1. Creer un repo public sur github.com  (ex: mon-iptv)
2. Uploader ce fichier : ma_liste_complete.m3u
3. Aller dans Settings > Pages > Branch main > Save
4. Ton lien final :  https://TON-USERNAME.github.io/mon-iptv/ma_liste_complete.m3u
